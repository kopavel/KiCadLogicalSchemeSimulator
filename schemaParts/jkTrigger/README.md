## JK-trigger

JK-trigger with R and S inputs.

### Pins

#### Input names:

- `J`- J input
- `K`- K input
- `C`- Clock
  sensitive to raising edge
- `R`- Reset
- `S`- Set

#### Output names:

- `Q`- output
- `~{Q}`- reverse output

### Parameters

#### Mandatory parameters:

- none

#### Optional parameters:

- `reverse`- `C` input sensitive falling edge.
