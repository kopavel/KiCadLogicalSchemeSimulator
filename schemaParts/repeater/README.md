## REPEATER

Buffer/inverter.

### Pins

#### Input names:

- `IN`- Input.

#### Output names:

- `OUT`- Output.

### Parameters

#### Mandatory parameters:

- none

#### Optional parameters:

- `reverse`- output reversed.
- `openColector`- open collector output.
- `openEmitter`- open emitter output.


### Example

NOT element: `reverse`
