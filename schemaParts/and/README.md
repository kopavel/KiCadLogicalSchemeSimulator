## AND gate

Implements logical AND gate functionality.

### Pins

#### Input names:

- `INx` Gate input  
  x — input number in range[0…63]

#### Output names:

- `OUT` Gate output

### Parameters

#### Mandatory parameters:

- `size` amount of input pins.

#### Optional parameters

- `reverse` reverse output for NAND gate.

### Example:

2-input NAND gate: `size=2;reverse`
