## LED

Interactive LED schema part.

### Pins

#### Input names:

- `A` anode
- `K` cathode

#### Output names:

- none

### Parameters

#### Mandatory parameters:

- none

#### Optional parameters:

- `size`- LED diameter on desk
- `onColor`- Turned on LED color in RGB #xxxxxx
- `offColor`- Turned off LED color in RGB #xxxxxx

### Example

`size=20;onColor=#FF0000;offColor=#808080`
