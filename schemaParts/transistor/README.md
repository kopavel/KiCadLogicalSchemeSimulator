## Bipolar Transistor

Logical equivalent of bipolar transistor.

### Pins

- `E`- Emitter.
- `K`- Collector.
- `B`- Base.

### Parameters

#### Optional parameters:

-`PNP`- implement PNP transistor, if provided. NPN otherwise.

Implemented as `Net filter`, which replace bipolar transistor with logically equivalent repeater.
